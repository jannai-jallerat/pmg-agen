Générateur d'icônes PWA — TPL

Cliquez sur "Générer et télécharger" pour créer les 3 icônes PNG nécessaires à la PWA.

icon-192.png
icon-512.png
icon-maskable.png


Étapes après téléchargement :

  1. Les 3 fichiers PNG sont dans votre dossier Téléchargements
  2. Dans votre projet, créez un dossier icons/ si absent
  3. Déplacez les 3 PNG dans icons/
  4. Faites un git add . && git commit -m "icônes TPL" && git push
